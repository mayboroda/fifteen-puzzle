# Fifteen Puzzle Game

## Prerequisite
- scala 2.12

## Project Structure
There are three modules in the project:

 Module     |  Decsription                                     
------------|-------------------------------------------------
 core       |  Game logic and implementation of game mechanics 
 game-cli   |  Command-Line puzzle implementation 
 game-fx    |  JavaFx puzzle implementation
